# Emir Otomat CNC - Backend

## Kurulum
```bash
npm install
```

## Çalıştırma
```bash
npm start
```

## Geliştirme Modu
```bash
npm run dev
```

## Environment Variables
Projeyi çalıştırmak için `.env` dosyası gereklidir. Örnek `.env` dosyası:
```properties
EMAIL_HOST=smtp.example.com
EMAIL_PORT=587
EMAIL_USER=user@example.com
EMAIL_PASS=your-password
...
```
